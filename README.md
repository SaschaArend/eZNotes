# eZNotes 🚀

**eZNotes** ist eine leistungsstarke Chrome-Erweiterung, mit der du im Handumdrehen professionelle Schritt-für-Schritt-Anleitungen und Prozessdokumentationen erstellen kannst.

![eZNotes Logo](assets/logo.png)

## ✨ Features

- **Automatisches Tracking**: Erfasst Screenshots und Metadaten (Sitzung, Feldname) bei jedem Klick.
- **Leistungsstarker Editor**:
    - **Markierungen**: Zeichne Rechtecke und Pfeile, um den Fokus zu lenken.
    - **Anonymisierung**: Verpixle sensible Daten direkt im Browser.
    - **Strukturierung**: Füge Kapitel und Textblöcke für eine klare Hierarchie hinzu.
- **Multi-Selection & Undo/Redo**: Effiziente Bearbeitung mehrerer Markierungen gleichzeitig und volle Kontrolle über alle Änderungen.
- **Export-Optionen**:
    - **Standard-Dokumentation**: Ein modernes, druckoptimiertes HTML-Format.
    - **Interaktive Tour**: Eine web-basierte Slideshow, die den Benutzer interaktiv durch den Prozess führt.
    - **Standalone-HTML**: Eine einzige, portable Datei für den einfachen Austausch.
- **Slate Design System**: Eine intuitive, moderne Benutzeroberfläche mit Support für Light & Dark Mode.

## 🚀 Installation (Developer Mode)

Da sich das Projekt aktuell in der Entwicklung befindet, kannst du es wie folgt manuell installieren:

1. Lade dieses Repository als ZIP herunter und entpacke es (oder nutze `git clone`).
2. Öffne Google Chrome und navigiere zu `chrome://extensions/`.
3. Aktiviere den **Entwicklermodus** (Schalter oben rechts).
4. Klicke auf **Entpackte Erweiterung laden** und wähle den Ordner aus, in dem die `manifest.json` liegt.
5. eZNotes erscheint nun in deiner Toolbar!

## 🛠 Benutzung

1. **Aufzeichnung starten**: Klicke auf das eZNotes-Icon und starte eine neue Session.
2. **Klicken**: Führe den Prozess durch, den du dokumentieren möchtest.
3. **Bearbeiten**: Öffne den Editor, passe Titel an, füge Markierungen hinzu oder verpixle sensible Daten.
4. **Exportieren**: Wähle dein gewünschtes Format aus und teile die Anleitung.
